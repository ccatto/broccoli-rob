Control Flow: {{ '@if' }} & {{ '@for' }}

{{ '@if' }} Directive Example

{{ '@if (showWelcome) {' }} {{ '}' }}

{{ '@for' }} Directive Example

{{ '@for (skill of angularSkills; track skill.id) {' }} then display with skill.name {{ '}' }}
Property Binding

Property binding allows you to bind component properties to HTML element properties.

{{ '{{ isEditable ? "Click here to edit this content!" : "Content editing is disabled" }}' }}
Event Handling 

        <p-button 
          label="Click Me!" 
          (click)="handleButtonClick()" 
          (mouseenter)="onMouseEnter()" 
          (mouseleave)="onMouseLeave()" 
          [style]="buttonStyle">
        </p-button>

        <p-message *ngIf="lastEvent" 
          [severity]="messageSeverity" 
          [text]="lastEvent">
        </p-message>
Input & Output Properties 

    <div class="example-container">
      <p>Demonstration of @Input and @Output decorators:</p>
      <p-inputText 
        [(ngModel)]="childMessage" 
        placeholder="Enter message for child component">
      </p-inputText>
      <app-child-demo 
        [message]="childMessage"
        (messageChanged)="onChildMessageChanged($event)">
      </app-child-demo>
      <p *ngIf="receivedMessage">
        Received from child: {{ '{{ receivedMessage }}' }}
      </p>
    </div>
